Tab Control ​

TabControl provides the user with a collection of tabs that can be used to display several contents.

TIP

TabStripPlacement property is available

Show ​

Example ​

xml
<TabControl>
    <TabItem Header="Tab 1">
        <!-- Content -->
    </TabItem>
    <TabItem Header="Tab 2">
        <!-- Content -->
    </TabItem>
</TabControl>
